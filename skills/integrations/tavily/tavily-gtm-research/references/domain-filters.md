# Tavily Domain Filters Guide

**Authoritative sources for high-quality research results**

---

## Why Use Domain Filtering?

**Benefits:**
- **Higher quality results** - Pre-vetted authoritative sources
- **Faster research** - Less time sorting through low-quality content
- **Better credibility** - Cite reputable sources in proposals and presentations
- **Consistent standards** - Team uses same trusted sources

**Time Savings:** 40% reduction in result review time

---

## How to Use Domain Filters

### Basic Syntax
```
Search Tavily for "query" only from domain.com
```

### Multiple Domains
```
Search Tavily for "query" only from domain1.com and domain2.com and domain3.com
```

### Exclude Domains
```
Search Tavily for "query" excluding spam-site.com
```

### Tier 1: Premium Sources (Highest Authority)
- Gartner, Forrester, IDC
- McKinsey, BCG, Bain
- Harvard Business Review
- MIT, Stanford research
- Government (.gov)

**Use When:** Need authoritative, citable sources for critical decisions

### Tier 2: Reputable Sources (High Quality)
- Bloomberg, WSJ, Reuters
- TechCrunch, Wired
- G2, TrustRadius
- LinkedIn, Crunchbase
- Industry associations

**Use When:** Need reliable, current information for general research

### Tier 3: Community Sources (Good Quality)
- Medium, Substack
- Stack Overflow, GitHub
- Reddit (specific subreddits)
- Industry blogs
- Company blogs

**Use When:** Need diverse perspectives or technical how-tos

## Domain Filter Strategies

### Strategy 1: Start Broad, Then Narrow
```
1. Search Tavily for "topic" (no filter)
2. Review results, identify quality domains
3. Re-search with domain filter for depth
```

### Strategy 2: Authority-First
```
1. Search Tavily for "topic" only from gartner.com and forrester.com
2. Get authoritative baseline
3. Expand to other sources if needed
```

### Strategy 3: Multi-Tier Approach
```
1. Tier 1: Search premium sources for framework
2. Tier 2: Search reputable sources for details
3. Tier 3: Search community for practical insights
```

## Measuring Filter Effectiveness

### Quality Indicators
- **Relevance**: Results match query intent
- **Authority**: Sources are credible
- **Recency**: Information is current
- **Completeness**: Covers all aspects
- **Actionability**: Insights are useful

### Track These Metrics
- Time to find relevant information
- Number of sources consulted
- Quality of insights (self-rated)
- Reusability of findings
