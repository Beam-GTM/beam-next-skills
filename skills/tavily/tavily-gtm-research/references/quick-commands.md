## Search Modifiers

### By Time Period
- `2026` - Current year
- `Q1 2026` - Specific quarter
- `January 2026` - Specific month
- `2025-2026` - Date range

### By Type
- `statistics` - Data and numbers
- `case study` - Real examples
- `best practices` - Guidelines
- `comparison` - Side-by-side analysis
- `review` - User feedback
- `guide` - How-to content

### By Quality
- `expert opinion` - Thought leadership
- `research report` - In-depth analysis
- `survey results` - Data-backed insights
- `whitepaper` - Comprehensive documents

- Paste into notes, CRM, or documents

- Create text snippets for frequent searches

## Troubleshooting Quick Fixes

### No Results?
- Broaden search terms
- Remove domain filters
- Try different keywords

### Irrelevant Results?
- Add more specific terms
- Use domain filtering
- Try advanced search depth

### Too Many Results?
- Add domain filters
- Use exact phrases in quotes
- Add time context (2026)

### Rate Limited?
- Check dashboard: https://tavily.com/dashboard
- Wait for monthly reset
- Upgrade plan if needed
