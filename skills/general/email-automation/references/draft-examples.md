# Email Draft Examples - Jonas Style

Quick reference for common email scenarios.

---

## 1. Quick Win Responses (< 2 min)

### Simple Yes
```
Hi [Name],

Yes, that works!

Looking forward.

-Jonas
```

### Simple No
```
Hi [Name],

Unfortunately that doesn't work for me, but [alternative] could work?

Let me know!

-Jonas
```

### FYI Acknowledgment
```
Hi [Name],

Got it, thanks for the heads up!

-Jonas
```

---

## 2. Scheduling Requests

### Meeting Request - Use Calendly
```
Hi [Name],

Happy to connect! Feel free to grab a time here: https://calendly.com/jonas

Looking forward to chatting.

-Jonas
```

### Meeting Request - Specific Times
```
Hi [Name],

Happy to chat! I'm free:
- Tuesday 2pm
- Wednesday 10am
- Thursday 3pm

Let me know what works!

-Jonas
```

### Reschedule Request
```
Hi [Name],

Can we push our call to [new date/time]? Something came up.

Thanks for understanding!

-Jonas
```

## 3. Introduction Responses

### Warm Introduction (Interested)
```
Hi [Name],

Thanks for the intro, [Introducer Name]!

[Name], great to connect. [1-2 sentences about interest/context]

Should we do a quick call next week? Here's my Calendly: [link]

Looking forward!

-Jonas
```

### Cold Outreach (Interested)
```
Hi [Name],

Thanks for reaching out! [Topic] sounds interesting.

[2-3 sentences about why you're interested or questions]

Let's do a call next week to discuss. Feel free to grab a time: [calendly]

-Jonas
```

### Polite Decline
```
Hi [Name],

Thanks for thinking of me! Unfortunately I'm focused on [current priority] right now, so I'll have to pass.

All the best with [their project]!

-Jonas
```

## 5. Follow-ups

### First Follow-up (After 1 Week)
```
Hi [Name],

Just wanted to follow up on this. Still interested in [topic]?

Let me know!

-Jonas
```

### Second Follow-up (After 2 Weeks)
```
Hi [Name],

Last ping on this! Let me know if you're still interested, otherwise no worries.

-Jonas
```

### After Meeting - Action Items
```
Hi [Name],

Great chatting today! Quick summary of next steps:

- [Your action item]
- [Their action item]
- [Joint action item]

-Jonas
```

## Template Variables

When generating drafts, replace these:
- `[Name]` → Recipient's first name
- `[Topic]` → Main subject/topic
- `[Date/Time]` → Specific date or time
- `[Link]` → Calendly or other link
- `[Specific thing]` → Be specific, not vague

**Key**: Always customize to the specific email context. These are patterns, not copy-paste templates.
